    Search Functionality Help

    You may search collection content in multiple ways.

    First you are presented with the global search bar at the top of the page. This will allow you to search through the entire database across all records for your query.

    You may modify and customize your search in a variety of ways.

    1. By using the “Fast Search” dropdown, you have the option to refine your immediate search in the following formats defined below.

      1. Fast Search | Searches the beginning of a string.

        Supports single character wildcards '_', for multi-character wildcards please use the Wildcard search option.

      2. Fuzzy Search | Database text search utilizing trigrams for terms.

        Performs an exhaustive search of the system for any strings similar to your input term. Will produce a large variety of results, however it will take longer to run.

      3. Wildcard Search | Searches for exact string in a term.

        You can use the '_' character for a single character wildcard or the '%' for a multi-character wildcard.

    2. On the left-hand side of this page, you have a pop-out button which allows you to filter on specific collection information returned in your initial search.

    3. You can click the to limit your search to specific sections of the database, for example if you only want to search taxonomies.
